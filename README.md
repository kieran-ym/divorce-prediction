# Divorce Prediction Project

This project predicts divorce outcomes using machine learning
based on survey response data.

## Dataset
- divorce_data.csv – survey responses
- reference.tsv – question descriptions

## Method
- Data cleaning
- Train-test split
- Logistic Regression / Random Forest
- Accuracy evaluation

## Results

- The best-performing model was **Logistic Regression**, achieving an
accuracy of **98.23%** on the test dataset.
